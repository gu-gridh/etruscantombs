# Etruscan Tombs

This is the repository for the Django application of the Etruscan Tombs project, led by Jonathan Westin at GRIDH. This project is developed as an app in Diana. It is developed as an initial clone of the [Jubileum portal](https://github.com/gu-gridh/jubileum).

It is corrently being developed by Matteo Tomasini and Tristan Bridge.
